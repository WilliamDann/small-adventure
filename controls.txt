Adventure Controls

General
-----------------
inventory : Manage your inventory
map       : View a map of the area

key:       List key of map characters and their meanings
help:      List commands
quit:      Quit Program

Movment
-----------------
up:    Move Up
down:  Move Down
left:  Move Right
right: Move Right

Each movment control can be shorthanded by using it's first character

Move into an NPC / Object to interact with it.
